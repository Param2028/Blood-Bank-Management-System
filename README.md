Blood-Bank-Management-System
The Blood Bank Management System is a web-based application designed to manage blood donors, patients, and blood inventory efficiently. The system helps blood banks maintain accurate records and quickly check the availability of different blood groups.
Users can register donors, manage patient information, and monitor blood stock through a simple and easy-to-use interface.

Features
Donor registration and management
Patient information management
Blood stock monitoring and updates
Database storage for records
Simple and user-friendly interface

Technologies Used
HTML
CSS
JavaScript
PHP
MySQL

Purpose
The aim of this project is to simplify the management of blood bank operations by maintaining organized records of donors, patients, and blood availability.
