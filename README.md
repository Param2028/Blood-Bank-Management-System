Blood Bank Management System
The Blood Bank Management System is a PHP & MySQL based web application to manage blood bank operations. It allows administrators to manage donor records, patient details, and available blood stock through a simple interface. The project uses core PHP for server‑side logic and HTML/CSS/JS for frontend.

Features
Add, update & view donor details
Manage patient information
Blood stock tracking
Login/logout functionality
Structured admin dashboard

Technologies Used
PHP
MySQL
HTML
CSS
JavaScript

Purpose
This system provides a digital solution for managing blood bank records, helping administrators maintain donor and patient data in a centralized database for efficient blood distribution and management.
